# Shareable website
A place to share and browse [Scriptable](https://scriptable.app) scripts and widgets.

## Core features
- <s>Authentication with GitHub</s>
- <s>Publish script</s>
- <s>Edit script</s>
- <s>Remove script</s>
- <s>Download script</s>
- <s>Home page</s>
- <s>Searching</s>
- Author pages

## Additional features
- Categories
- Report script
- Rate script
- Star script
- Discord integration
